# Лабораторна робота №2 - JavaScript Web-додаток TimeFlow

## Інформація про студента
- **ПІБ:** Понамарьова Сніжана
- **Група:** КВ-34

## Назва лабораторної роботи
**Лабораторна робота №2: Розробка функціональності Web-додатка засобами JavaScript**

## Завдання
Розробити JavaScript-версію Web-додатка **TimeFlow**:
- реалізувати функціональність основних сторінок додатка;
- організувати код відповідно до шаблону **MVC**;
- створити моделі, представлення та контролери для сторінок додатка;
- реалізувати логіку авторизації, реєстрації, профілю користувача та сторінки трекера;
- забезпечити збереження даних користувача та робочих сесій за допомогою **localStorage**;
- підготувати діаграму класів JavaScript-додатка та оновлений звіт.

## Реалізовані сторінки
- **Головна** (`index.html`)
- **Про додаток** (`about.html`)
- **Трекер** (`tracker.html`)
- **Профіль** (`profile.html`)
- **Вхід** (`login.html`)
- **Реєстрація** (`register.html`)

## Використані технології
- **HTML5**
- **CSS3**
- **Bootstrap 5**
- **JavaScript (ES6)**
- **localStorage**
- **MVC**
