## Назва лабораторної роботи

**Лабораторна робота №3:** Розробка Web-додатка засобами фреймворку VueJS

## Завдання

Розробити VueJS-версію Web-додатка **TimeFlow** з клієнт-серверною архітектурою:

- реалізувати клієнтську частину додатка засобами VueJS;
- реалізувати основні сторінки Web-додатка: головну сторінку, сторінку опису, трекер, профіль, вхід та реєстрацію;
- забезпечити реактивну роботу інтерфейсу за допомогою VueJS;
- реалізувати реєстрацію та авторизацію користувача;
- організувати обмін даними між клієнтською та серверною частиною через асинхронні HTTP-запити;
- створити серверну частину за допомогою Node.js та Express;
- забезпечити збереження користувачів, токенів авторизації та робочих сесій у базі даних SQLite;
- реалізувати роботу таймера: старт, пауза, продовження, зупинка та збереження сесії;
- підготувати звіт та матеріали для захисту лабораторної роботи.

## Використані бібліотеки та модулі

### Клієнтська частина

- **VueJS 3** — використовується для створення реактивного інтерфейсу, роботи з формами, кнопками, таблицею сесій і таймером.
- **Bootstrap 5** — використовується для стилізації сторінок, форм, кнопок, таблиць та адаптивної верстки.
- **Fetch API** — використовується для асинхронної взаємодії клієнта із сервером.
- **localStorage** — використовується для збереження токена авторизації у браузері.

### Серверна частина

- **Express.js** — використовується для створення сервера, API-маршрутів і обробки HTTP-запитів.
- **cors** — використовується для налаштування доступу до серверного API.
- **sqlite3** — використовується для роботи з базою даних SQLite.
- **path** — вбудований модуль Node.js для роботи зі шляхами до файлів.
- **crypto** — вбудований модуль Node.js для генерації токенів авторизації.
