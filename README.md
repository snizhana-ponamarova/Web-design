## Завдання
Розробити статичну версію Web-додатка **TimeFlow**:
- створити набір сторінок (головна, про додаток, трекер, профіль, вхід, реєстрація);
- реалізувати навігацію між сторінками;
- оформити інтерфейс із використанням HTML/CSS та Bootstrap;
- забезпечити адаптивність для різних розмірів екрана (смартфон/планшет);
- підготувати звіт із скріншотами та структурною схемою сайту.
